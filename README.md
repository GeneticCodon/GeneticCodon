## What We Build

- Gene and disease data tools powered by public APIs (Ensembl, Open Targets)
- Streamlit web apps for researchers who prefer a UI over the command line
- Tools that remove the manual lookup work from standard bioinformatics workflows

---

## Data Sources

All tools use openly accessible, no-key-required APIs:

- [Ensembl REST API](https://rest.ensembl.org) — gene annotation and orthology
- [Open Targets Platform](https://platform.opentargets.org) — disease associations and drug target tractability
